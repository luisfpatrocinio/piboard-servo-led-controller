# PiBoard - Servo & LED Controller

This is a simple project for the PiBoard that controls a servo motor and three LEDs (Green, Yellow, Red) based on button presses. 

## Features
- Cycles through 3 states using a single button.
- Updates an LED indicator and a Servo motor angle according to the state.
- Includes debouncing for reliable button reads.

## Hardware Setup
- **Button Pin:** 2
- **Green LED Pin:** 4
- **Yellow LED Pin:** 12
- **Red LED Pin:** 13
- **Servo Motor Pin:** 11

Built with [PlatformIO](https://platformio.org/).
